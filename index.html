Polygon Overlap Calculator

This project demonstrates a solution to a programming puzzle. Given two polygon vertices, compute the area of their overlap.

Sort vertices clockwise

To sort a polygon's vertices clockwise, we need any inner point, which could be a vertex or the centroid. This will be the reference point. Then for every vertex, we calculate the polar angle from the reference point to the vertex relative to the x-axis. Thus, we can order the vertices by their calculated angle.
