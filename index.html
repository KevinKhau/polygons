Sorting vertices clockwise, using a ref and polar angles

To sort a polygon vertices clockwise, we need any inner point, which could be also be a vertex or the centroid. This will be the reference point. Then for every vertex, we calculate the polar angle between the x-axis and the vertex to the reference point. Thus we can order the vertices according to their found angle. You can here.
